# SEOBoost Design Guidelines

Inspired by the supplied Hexon reference:

- Dark, high-contrast SaaS security feel translated into SEO escrow.
- Neon green primary accent for verified white-hat performance.
- Centered hero, pill navigation, dashed trust badge, network/ranking graph motifs.
- No inner page scrollbar. The browser/document is the only vertical scroll container.
- Product surface should feel like an audit war room, not a generic intake form.
